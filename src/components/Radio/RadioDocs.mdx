The `Radio` component is a component that allows users to select one option from a list of options. It is a part of the `RadioGroup` component.

### Import 
```ts
import { Radio } from '@component/Radio/Radio';
```

### API
*Note: **Click onto the component name** to view its props and events document*
| Component | Description |
| - | - |
| [`<Radio>`](#api) | Sub-component used to render the radio options, wrap by `<RadioGroup>` component. |
| [`<RadioGroup>`](#api) | Primary component is the component used to render set of user options |
