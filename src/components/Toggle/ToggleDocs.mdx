The `Toggle` component is a simple way to toggle between two states.

### Import 
```ts

```

### Accessibility
#### **Keyboard Support**
| Key   | Function |
| :---- | :----    |
| Tab | Moves keyboard focus to the toggle. |
| Space | Toggle between checked and unchecked states |
#### **Role, Property, State, and Tabindex Attributes**
Attribute | Value |
| :---- | :----    |
| role  | `switch` |
| aria-checked | `checked state of component` |
| aria-disabled | `disabled state of component` |
